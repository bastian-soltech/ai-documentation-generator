# Proyek Matematika
## Deskripsi
Proyek ini berisi kode untuk melakukan operasi matematika sederhana.
## Teknologi yang Digunakan
- Python 3.x
## Cara Instalasi
1. Pastikan Python 3.x sudah terinstal di komputer Anda.
2. Unduh kode sumber dari repository ini.
## Cara Penggunaan
1. Buka terminal atau command prompt.
2. Navigasikan ke direktori tempat kode sumber disimpan.
3. Jalankan kode dengan perintah `python math.py`.
## Fungsi yang Tersedia
- `kurangi_bilangan(a, b)`: Mengurangi dua bilangan dan mengembalikan hasilnya.
